
# Customer Purchase Prediction using Machine Learning

## Overview
This project predicts whether a customer will purchase a product using demographic and behavioral data.

## Tools Used
- Python
- Google Colab
- Pandas, NumPy
- Scikit-learn
- Excel

## Dataset
- 10,000 customer records
- Features: Age, Monthly Income, Visits per Month
- Target: Purchased (Yes/No)

## Model
- Logistic Regression (Classification)

## Workflow
1. Load Excel data
2. Train ML model
3. Generate predictions
4. Save results back to Excel

## Business Use
The prediction output can be shared with sales and marketing teams to prioritize high-potential customers.
